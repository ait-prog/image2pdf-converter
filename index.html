Image to PDF Converter

An advanced tool for converting images to PDF with both GUI and CLI interfaces.

Features

  • Convert multiple images to a single PDF
  • Graphical interface with drag and drop support
  • Command line interface for batch processing
  • Adjustable quality and DPI settings
  • Supports PNG, JPG, JPEG, BMP, WEBP, TIFF formats

Installation

pip install image2pdf-converter

Usage

GUI Mode:

image2pdf-gui

CLI Mode:

image2pdf image1.jpg image2.png -o output.pdf --quality 90 --dpi 300
